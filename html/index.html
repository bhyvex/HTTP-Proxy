Presentation

HTTP::Proxy is a pure Perl RFC 2616 compliant proxy module. At least, that's my goal.

Read the module documentation.

Features

HTTP::Proxy version 0.10 supports the following features:

  • Persistent HTTP/1.1 connections
  • Filter chains on the requests and response, applied according to the scheme, method, host, path or MIME type
  • Helper filters that make writing new filters a very easy task

Warning: As from version 0.10, a new API for filter exists, which is not compatible with previous versions of HTTP::Proxy.

HTTP::Proxy usage

Other people or projects actually use HTTP::Proxy:

  • HTTP::Recorder, (by Linda Julien) is a User-Agent designed to be used by HTTP::Proxy to record interaction with web sites and create WWW::Mechanize scripts.
  • HTTP::Proxy::BodyFilter::ADnix (by Cosimo Streppone) is a filter designed to block advertising images with custom regexps.

HTTP::Proxy in the press

  • June 2004: There's an article about HTTP::Recorder by Linda Julien on perl.com.
