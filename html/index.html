Presentation

HTTP::Proxy is a pure Perl RFC 2616 compliant proxy module. At least, that's my goal.

Read the module documentation.

Features

HTTP::Proxy version 0.07 supports the following features:

  • Persistent HTTP/1.1 connections
  • Filter chains on the requests and response, applied according to the scheme, method, host, path or MIME type

Support

There is a developper mailing-list, http-proxy[at]mongueurs.net, where you can ask questions about this module, submit patches and receive all the commit emails.

To subscribe, send an email to majordomo@mongueurs.net with the line subscribe http-proxy in the body of your message.
