HTTP::Proxy::BodyFilter

  • This is the base class for manipulating message bodies

  • The filter() method has the following signature:

        sub filter {
        my ( $self, $dataref, $message, $protocol, $buffer ) = @_;
        ...
    }

  • In the case of response filters, the headers ($message->headers()) have already been sent to the client, so technically they're read-only.
