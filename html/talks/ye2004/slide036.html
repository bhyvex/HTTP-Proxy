Tools using HTTP::Proxy

  • HTTP::Recorder, by Linda Julien

    The first version of HTTP::Recorder was using filters.

    The current implementation is actually a subclass of LWP::UserAgent that records what the proxy asks for and outputs Perl code that runs a WWW::Mechanize robot.

  • HTTP::Proxy::BodyFilter::Adnix, by Cosimo Streppone

    This is a simple ad-blocker. When receiving an image (image/*) from a known ad-sending site, it replaces it by another image.
