There are other web proxies in Perl

Oh yes. And written by gurus.

  • http://www.stonehenge.com/merlyn/WebTechniques/col11.html

    Anonymiser written by Randal L. Schwartz. Removes cookies, From and Referer.

  • http://www.foad.org/~abigail/Perl/proxy.pl

    Another anonymiser, written by Abigail.

  • http://rgarciasuarez.free.fr/perl/biscuit

    This proxy records cookies and lets you remove/edit them on a dedicated page (generated by the proxy).

  • http://www.research.att.com/~hpk/wsp/

    The Web-scraping proxy, is a powerful tool for web scraping. Version 2 supports HTTPS (performing a man in the middle attack).

  • http://www.jmarshall.com/tools/cgiproxy/

    It is actually a CGI script that returns the pages you ask for. This is not a proxy.
